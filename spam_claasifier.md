
## Spam Classifier Thinking
**Problem Statement:**  
Design a system to detect whether a message is spam or not.

---

**Features (Input Data):**  
- Message text  
- Keywords (e.g., "win", "free", "offer")  
- Number of links  
- Sender information  
- Message length  

---

**Data Needed:**  
- A dataset of messages  
- Each message labeled as:
  - Spam  
  - Not Spam  

---

**Output:**  
- Spam  
- Not Spam  

---

**How Machine Learning Works:**  
The model learns patterns from spam messages and classifies new messages based on those patterns.

---

**Possible Mistakes:**  
- False Positive: Normal message marked as spam  
- False Negative: Spam message marked as normal  

---

**Examples:**  
- "Win ₹1000 now!!!" → Spam  
- "Meeting at 5 PM" → Not Spam  

---

**Conclusion:**  
Spam classifiers help filter unwanted messages and improve user experience.
